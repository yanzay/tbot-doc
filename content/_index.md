# tbot - Telegram Bot Server

## Features

- Full Telegram Bot API **4.4** support
- **Zero** dependency
- Type-safe API client with functional options
- Capture messages by regexp
- Middlewares support
- Can be used with go modules
- Support for external logger
- MIT licensed
